stitched.eth Garment Registry

  • How to tell if a garment is authentic?
    • Method 1. Scan the NFC on the garment. The NFC should link to somewhere on this page with an image of the garment and a lot of other information, including the STX blockchain address. If the garment has a tye-dye tag, you can compare the photos of the tag on the garment with the one on this website and "eyeball" it to tell whether or not the garment is authentic. However, to establish final proof of authenticity, use Method 2.
    • Method 2. The owner of the garment can connect with the app on the STX NFT Verification URL (https://boom.money/home) using a STX web3 compatible browser like Xverse for mobile or Brave on desktop with the Hiro Wallet plugin. An image of the garment will be listed on that page as one of their NFTs. The NFT will have an image and list a Creator and Name. Make sure that both the Name of the NFT and the Creator Address match the name listed on this website. 
  • How does all of this work? As soon as the garment is manufactured, an NFT is created with a photo of the garment and the tag. This NFT will be transferred to the person who buys the garment. The owns the garment can prove authenticity of the garment by establishing they own the STX NFT listed on this page. This is the only way to establish that the garment is authentic. If you are considering purchasing the garment, you must ensure that you receive both the garment and the corresponding NFT. If you purchase a garment without getting the corresponding NFT, there will be no way for you to prove with absolute certainty that the garment is authentic.

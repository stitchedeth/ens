Governance / $THREAD

The $THREAD token is the governance token of stitched.eth. The $THREAD token is on Polygon at 0x64138094AFAe67c9796E2C9d250C977e3D813573. You can acquire $THREAD tokens by participating in the the stitched.eth ecosystem. 

Voting is currently fone on Snapshot, but this will ultimately migrate to an on-chain voting platform. 

All $THREAD token holders will be allowed to vote on each proposal. The stitched.eth community should endeavor to implement the votes of the $THREAD token holders. 

This decentralized website is powered by the Ethereum Name Service and built on IPFS.
